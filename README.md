# sketch_may27a

M5PaperColor 用の名札表示スケッチです。  
上部に画像、下部に温度・湿度を表示します。

## 機能

- SD カード内の PNG / JPG / JPEG / BMP を表示
- SD に画像がない場合は、スケッチ内蔵のデフォルト画像を表示
- 画像は表示枠内にアスペクト比を保って自動縮小
- 左右スワイプで SD 内画像を切り替え
- SHT40 の温度・湿度を表示
- 温湿度は起動時に取得し、その後は 30 分ごとに更新
- 更新速度優先で、画像領域と下部情報領域を分割更新

## 対象ハードウェア

- M5PaperColor
- SHT40
- SD カード

## 使用ライブラリ

- M5Unified
- M5GFX
- M5PM1
- M5UnitENV
- SD

## 表示レイアウト

画面サイズは 400 x 600 を前提にしています。

- 画像の実表示領域: 352 x 358 px
- 温度・湿度カード: 下部 2 カラム

## 画像の優先順位

1. SD カード直下の対応画像
2. スケッチに埋め込まれたデフォルト画像

SD 内の画像は最大 32 件まで読み込みます。

## センサ取得

SHT40 は以下の候補を順に探索します。

1. In_I2C (GPIO2 / GPIO3), address 0x44
2. In_I2C (GPIO2 / GPIO3), address 0x45
3. Ex_I2C (GPIO5 / GPIO4), address 0x44
4. Ex_I2C (GPIO5 / GPIO4), address 0x45

シリアルモニタには検出結果と更新結果を出力します。

## ファイル構成

- `sketch_may27a.ino` - メインスケッチ
- `default_image.h` - 埋め込み済みデフォルト画像

## ビルドと書き込み

1. Arduino IDE でこのフォルダを開く
2. ボードに **M5PaperColor** を選ぶ
3. シリアルポートを選ぶ
4. 通常のスケッチ書き込みを実行する

LittleFS への追加アップロードは不要です。  
デフォルト画像は `default_image.h` に埋め込まれています。

## SD カード画像の置き方

SD カードのルートに画像ファイルを置いてください。

対応拡張子:

- `.png`
- `.jpg`
- `.jpeg`
- `.bmp`

## メモ

- 表示モードは `epd_fastest`
- 温湿度の再取得間隔は `30UL * 60UL * 1000UL`
- SD がなくてもデフォルト画像で動作します
